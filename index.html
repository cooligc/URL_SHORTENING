Welcome Url Shortening Application.

This is an app where we can short our url by passing some parameter to the Server. And we'll get a new URL which will be having some hash key. If we'll hit the new URL in the browser, Our REST Service will call and redirect the URL to the original URL.

We have different branches present for this app . You can use the following commands to check it out.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Prerequisite to run this app

  • Java 7 or above
  • Maven 3 or above
  • MongoDB 2.6 or above
  • Tomcat 7 or above
  • Any REST Client either command line REST Client like cURL or visual REST Client like POSTMan or Advanced REST Client

Technology used

  • Spring v4.0.6.RELEASE
  • Jersey v2.11
  • Mongo Driver v2.10.1

How to run this application

  • Clone the repository
  • Build the application using maven (mvn clean install)
  • Take the war file from target folder and deploy it on the server
  • Start the Server

Now app will be running.
